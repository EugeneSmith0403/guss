# Frontend - The Last of Guss

Браузерная игра, где игроки соревнуются, кто быстрее и больше натапает по мутировавшему виртуальному гусю G‑42.

## Технологический стек

- **React + TypeScript** — фундамент приложения
- **React Hook Form + Zod** — валидация форм
- **React Query** — сетевые запросы и кэширование
- **React Router** — маршрутизация
- **Material UI + TailwindCSS** — UI компоненты и стилизация
- **Zustand** — стейт‑менеджер
- **react-i18next** — интернационализация (RU/EN)
- **Vite** — сборка

## Установка

```bash
pnpm install
```

## Запуск

```bash
pnpm dev
```

Приложение будет доступно на `http://localhost:8080`

## Переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:3000
```

## Структура проекта

```
/src
 ├─ api/              # API клиенты
 ├─ components/        # React компоненты
 ├─ pages/            # Страницы приложения
 ├─ stores/           # Zustand stores
 ├─ i18n/             # Локализация
 ├─ router/           # Маршрутизация
 └─ workers/          # Service Worker
```

## Основные страницы

- `/login` — авторизация/регистрация
- `/rounds` — список раундов
- `/rounds/:id` — страница раунда с кликабельным гусем

## Функциональность

- ✅ Авторизация с автоматической регистрацией
- ✅ Интернационализация (RU/EN)
- ✅ Список раундов с автоматическим обновлением
- ✅ Кликабельный SVG гусь с анимацией
- ✅ Offline режим с синхронизацией
- ✅ Таймеры обратного отсчета
- ✅ Кнопка выхода

## Особенности

- **Offline режим**: клики сохраняются локально и синхронизируются при восстановлении сети
- **Анимация клика**: гусь подсвечивается при клике с задержкой 200ms и троттлингом
- **Автоматическое обновление**: список раундов обновляется каждые 5 секунд
- **Локализация**: автоматическое определение языка браузера
